# 📊 Data Visualization Project

## 📌 Overview
This project analyzes student performance using Python and visualizes data using different charts.

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib

## 📈 Visualizations
- Line Chart → Study Hours vs Marks
- Bar Chart → Student Marks Comparison
- Histogram → Marks Distribution

## 💡 Key Insights
- Students who study more score higher marks
- Performance varies among students
- Most students score in the mid range

## 📂 Files Included
- main.py → Python code
- data.csv → Dataset
- line_chart.png → Line graph
- bar_chart.png → Bar graph
- histogram.png → Histogram

## 🚀 Conclusion
Data visualization helps in understanding patterns and making better decisions.
